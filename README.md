# URL Shortener

This is a simple URL Shortener backend application built using Node.js, Express, and MongoDB Atlas.

The application generates a short URL for a given long URL and redirects users to the original URL.

---

## Features

- Create short URLs
- Redirect to original URLs
- MongoDB Atlas integration for persistent storage
- Environment-based configuration using dotenv

---

## Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
